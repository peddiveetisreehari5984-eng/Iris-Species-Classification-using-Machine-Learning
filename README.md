# 🌸 Iris Species Classification

A simple Machine Learning project using the built-in Iris dataset.

## Goal
Predict flower species based on measurements.

## Algorithm
Random Forest Classifier

## Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## How to Run

### Install dependencies
pip install -r requirements.txt

### Train model
python src/train.py

### Run predictions
python src/predict.py

## Output
- Model accuracy
- Classification report
- Confusion matrix
- Interactive prediction system
